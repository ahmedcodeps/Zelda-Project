# Zelda-Project
Zelda project showcase, make in LOVE using Lua with VS Code.

## Credits:
Credits to Mister Mike for all the sprites used.
Credits to Sheepolution for guides on collision and game mechanics

## How to run:
1. Download LÖVE from https://love2d.org
2. Use the following command on Bash:
Mac: ~/path/to/love ~/path/to/your-project
Windows: "C:path/to/LOVE/love.exe" project.
